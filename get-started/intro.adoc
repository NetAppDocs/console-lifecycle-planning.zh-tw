= 了解生命週期規劃
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/get-started/


[role="lead"]
生命週期規劃評估目前或預測的低容量儲存資產，並提供滿足您業務需求的資料分層選項建議。



== NetApp控制台

可透過NetApp控制台存取生命週期規劃。

NetApp控制台提供企業級跨本機和雲端環境的NetApp儲存和資料服務的集中管理。需要控制台才能存取和使用NetApp資料服務。作為管理介面，它使您能夠從一個介面管理許多儲存資源。控制台管理員可以控制企業內所有系統的儲存和服務的存取。

您不需要許可證或訂閱即可開始使用NetApp控制台，並且只有當您需要在雲端部署控制台代理程式以確保與儲存系統或NetApp資料服務的連接時才需要付費。但是，一些可從控制台存取的NetApp資料服務是需要授權或基於訂閱的。

詳細了解 https://docs.netapp.com/us-en/bluexp-setup-admin/concept-overview.html["NetApp控制台"]。



== 您可以使用生命週期規劃做什麼？

生命週期規劃可以幫助您實現以下目標。

*容量資訊*

* 識別環境中容量較低的儲存資產。
* 審查資產的當前和預測容量。




== 生命週期規劃的好處

生命週期規劃有以下優勢：

* 識別即將發生生命週期事件的系統，以協助您規劃技術更新
* 提供值得信賴的顧問的建議來解決您環境中的容量問題
* 減少規劃額外儲存所花費的時間




== 成本

NetApp不會向您收取使用生命週期規劃的費用。

如果您決定將冷資料分層到雲端存儲，則需要檢查雲端提供者是否收取適用的費用。



== 使用權

您可以透過NetApp控制台（*儲存* > *生命週期規劃*）或透過Active IQ Digital Advisor （也稱為Digital Advisor）（*儀表板* > *規劃小工具* > *技術更新*）存取此功能。
